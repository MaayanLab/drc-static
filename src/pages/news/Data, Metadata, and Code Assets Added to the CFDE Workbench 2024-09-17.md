title: Data, Metadata, and Code Assets Added to the CFDE Workbench

New digital assets were recently added to the CFDE Workbench by the Common Fund DCCs. These include updated C2M2 metadata packages from GlyGen, IDG, HuBMAP, Metabolomics, SPARC, and Kids First; ETL and KG assertions from Bridge2AI; and an updated XMT file from LINCS.
